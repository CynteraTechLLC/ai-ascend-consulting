# CynteraTech LLC Platform

Enterprise AI consulting and engineering. We build the intelligence behind your business — from strategy to production deployment.

## Development

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:8080](http://localhost:8080) with your browser to see the result.

## Build

To build the project for production:

```bash
npm run build
```
